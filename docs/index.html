注意事項

            
適用範囲

          

        

        
    
          
  • PC と XBee の設定通信は、XBee の現在の UART ボーレートを候補から自動検出して行います。
    • 
          
  • 通常の1対1ペアリングで変更する設定は ID、CE、DL、BD です。BD は XBee 同士が接続機器と通信するボーレートです。
    • 
          
  • API モードでは 3 台以上を対象に ID、CE、BD、AP=1 を書き込みます。DL は変更しません。
    • 
          
  • SL は読み取り専用です。通常の1対1ペアリングでは相手側の SL を自動で DL に書き込みます。
    • 
          
  • DH は変更しません。既存値が 0 以外だと通信できない可能性があります。
    • 
          
  • Function Set / Firmware は変更しません。ZIGBEETHReg のまま使ってください。
    • 
          
  • BD や AP の変更は ATWR 書き込み後に XBee を再起動（電源 OFF/ON）すると有効になります。
